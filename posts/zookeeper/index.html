Zookeeper

 2019.7.7  2021.7.27  1474  3 分钟

Zookeeper是分布式系统中常用地分布式协调服务，它给出了基本的原语操作，分布式程序可以基于这些原语构建更多高级的功能，比如服务的同步，配置管理，以及分布式锁等。

安装

官方网站下载最新稳定版编译好的tar.gz文件，解压文件，然后修改配置，分别安装到四台主机上。默认已经安装JDK

      [root@zookeeper-node04 conf]# cat zoo.cfg
  # The number of milliseconds of each tick
  tickTime=2000
  # The number of ticks that the initial 
  # synchronization phase can take
  initLimit=10
  # The number of ticks that can pass between 
  # sending a request and getting an acknowledgement
  syncLimit=5
  # the directory where the snapshot is stored.
  # do not use /tmp for storage, /tmp here is just 
  # example sakes.
  dataDir=/var/zk
  # the port at which the clients will connect
  clientPort=2181
  # the maximum number of client connections.
  # increase this if you need to handle more clients
  #maxClientCnxns=60
  #
  # Be sure to read the maintenance section of the 
  # administrator guide before turning on autopurge.
  #
  # http://zookeeper.apache.org/doc/current/zookeeperAdmin.html#sc_maintenance
  #
  # The number of snapshots to retain in dataDir
  #autopurge.snapRetainCount=3
  # Purge task interval in hours
  # Set to "0" to disable auto purge feature
  #autopurge.purgeInterval=1

  ## Metrics Providers
  #
  # https://prometheus.io Metrics Exporter

  server.1=zookeeper-node01:2888:3888
  server.2=zookeeper-node02:2888:3888
  server.3=zookeeper-node03:2888:3888
  server.4=zookeeper-node04:2888:3888

  • 创建myid文件，设置节点的id值，对应以上server.4中的数字

    1
2

    [root@zookeeper-node04 conf]# mkdir /var/zk
[root@zookeeper-node04 conf]# echo 4 > /var/zk/myid

存放数据文件配置为/var/zk目录，添加4台机器端口配置在下面。

  • 设置zookeeper的环境变量，修改/etc/profile，加入对应路径配置： 
    1
2
3
4

    export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.292.b10-1.el7_9.x86_64
export ZOOKEEPER_HOME=/usr/local/zookeeper
export CLASSPATH=.:$JAVA_HOME/jre/lib/rt.jar:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin:$ZOOKEEPER_HOME/bin
  • 将zookeeper及其对应配置直接复制到其他三台机器上 [root@zookeeper-node01 local]# scp -r ./zookeeper/ zookeeper-node01:pwd [root@zookeeper-node01 local]# scp /etc/profile zookeeper-node01:/etc
  • 修改对应机器的myid 
    1

    [root@zookeeper-node04 conf]# echo 1 > /var/zk/myid
  • 启动4台机器的zookeeper，在每台机器上运行命令： 
    1

    [root@zookeeper-node04 conf]# zkServer.sh start-foreground
